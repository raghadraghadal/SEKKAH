# 🚆 SEKKAH – Railway Management System

SEKKAH is a web-based railway management system designed to help users monitor and manage train operations efficiently. The system provides multiple features for handling trains, schedules, and passenger-related activities.

---

## 📌 Features

* Train monitoring page with organized data display
* View train schedules and timetables
* Add new trains with validation (prevent duplicate train numbers)
* Passenger management interface
* Reservation system interface
* Train reports and information pages
* User authentication pages (Login / Sign Up / Forgot Password)
* Profile management

---


##  How to Run

1. Clone the repository:
   git clone https://github.com/raghadraghadal/SEKKAH.git

2. Open the project folder

3. Open the file `home.html` using any web browser (Google Chrome, Edge, etc.)

The system will run locally for demonstration purposes.
